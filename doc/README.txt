Ziel dieses Tool ist es Metadaten (Autor, Bewertung, Abzahlbewertung, Genres) zu bestimmten Bücherlisten lokal zu Speichern. Anschließend kann überprüft werden, ob vergleichbaer Bücher lokal vorhanden sind.
Für die Listen wird Goodreads verwendet.

fill_best_books.py -- Lädt Bücherlisten von Goodreads herunter und speicher diese lokal ab.

compare_goodreads_with_local_books --Vergleich den lokal Buchbestand mit den herungerladenen Metadaten

content/input.py -- bestimme, welche Listen heruntergalden werden
